🔄 FLUJO DE TRABAJO GIT
════════════════════════════════════════════════════════════════════════════

Para trabajar con el repositorio:

1. Descargar cambios desde GitHub:
   $ git fetch origin
   $ git pull origin main

2. Hacer cambios locales:
   $ git add archivo.php
   $ git commit -m "feat: Descripción del cambio"

3. Enviar cambios a GitHub:
   $ git push origin main

4. Ver historial:
   $ git log --oneline

5. Ver estado actual:
   $ git status

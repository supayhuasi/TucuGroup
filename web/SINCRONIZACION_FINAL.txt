📥 CÓMO CLONAR EL PROYECTO
════════════════════════════════════════════════════════════════════════════

git clone https://github.com/supayhuasi/TucuGroup.git
cd TucuGroup/web
composer install
npm install
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve

Luego accede a: http://localhost:8000/admin/dashboard
Email: admin@example.com
Password: password

GUÍA RÁPIDA - MÓDULO DE ADMINISTRACIÓN
======================================

1️⃣ INSTALACIÓN RÁPIDA
──────────────────────

```bash
# Ejecutar migraciones
php artisan migrate

# (Opcional) Crear usuarios de prueba
php artisan db:seed
```

2️⃣ ACCESO AL PANEL
───────────────────

Después de instalar, inicia sesión con:

Email: admin@example.com
Contraseña: password

Luego ve a: http://tu-app.local/admin/dashboard

3️⃣ LO QUE PUEDES HACER
──────────────────────

✨ Dashboard         → Ver estadísticas de usuarios
👥 Gestionar         → Ver, editar, eliminar usuarios
🔑 Cambiar Roles     → Hacer admin o quitar admin
⚙️  Configuración    → Ver info del sistema

4️⃣ CARACTERÍSTICAS PRINCIPALES
───────────────────────────────

✅ Panel de control con estadísticas en tiempo real
✅ Gestión completa de usuarios
✅ Sistema de roles (admin/usuario)
✅ Control de acceso protegido
✅ Interfaz responsive y dark mode
✅ Validaciones de seguridad

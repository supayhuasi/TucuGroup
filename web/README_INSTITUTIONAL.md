## 🚀 Cómo Usar

### Instalación
```bash
cd /home/rodrigo/TucuGroup/TucuGroup/web

# Opción rápida
bash setup-institutional.sh

# O manual
composer install && npm install && npm run build
```

### Desarrollo
```bash
php artisan serve
npm run dev
# Acceder a: http://localhost:8000
```

### Producción
```bash
npm run build
php artisan config:cache
# Deployar a servidor
```

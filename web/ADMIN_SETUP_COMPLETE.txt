🚀 PARA EMPEZAR
────────────────────────────────────────────────────────────────────────────

1. Ejecutar migraciones:
   $ php artisan migrate

2. (Opcional) Crear usuarios de prueba:
   $ php artisan db:seed

3. Iniciar servidor:
   $ php artisan serve

4. Acceder al panel:
   ✓ Ir a http://localhost:8000/login
   ✓ Ingresar con: admin@example.com / password
   ✓ Hacer clic en "🔐 Administración"
   ✓ ¡Listo! Ya estás en el panel

════════════════════════════════════════════════════════════════════════════

📖 RUTAS DISPONIBLES EN EL PANEL
────────────────────────────────────────────────────────────────────────────

GET  /admin/dashboard           → Dashboard
GET  /admin/users               → Lista de usuarios
GET  /admin/users/{id}/edit     → Editar usuario
PATCH /admin/users/{id}         → Actualizar usuario
DELETE /admin/users/{id}        → Eliminar usuario
POST /admin/users/{id}/toggle-admin → Cambiar rol
GET  /admin/settings            → Configuración
